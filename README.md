# Journal App

This is a react project to strengthen knowledge, the following technologies were used

1. Vite - React
2. Redux Toolkit
3. Firebase

# commands

1. clone
2. installs dependencies
```
npm install
```
3. enviromment dev
```
npm run dev
```
4. building for production
```
npm run build
```
5. preview production (after the of build)
```
npm run preview
```

Good look :)
